HTML

  • HTML is used to create the structure of a webpage and is made up of elements.
  • There are Text, Header, Image, and List elements.
  • Elements are usually composed of an opening tag and a closing tag (< and >)
  • The Head element contains information about the webpage (Metadata).
  • Elements included in the Head are not visible when the user looks at the webpage.
  • The Body element represents the visible content shown to the user.
  • HTML elemnts are case sensitive.
  • Elements nestled inside other elements are called child elements. The element they are nestled in is called the parent element.
  • It is best practice to use Semantic HTML elements as these describe the content and the role the elements play on the webpage.
  • Element and Tag are often used interchangeably.

CSS

  • CSS is used to style a webpage.
  • There are 3 ways to style using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Inline CSS: uses the style atttribute to change a single element
  • Internal CSS Style Sheet: embeds an entire stylesheet directly in the HTML file using a style element in the head element
  • External CSS Style Sheet: keeps all your CSS rules in a separate file by using a link element in the HTML head element
  • External CSS Style Sheet is the most common and preferred way to style a webpage.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a version control system.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: takes you back to the main branch
  • git pull origin main: pulls down the latest and most updated main branch
  • git add -A: commits all files in the working branch
  • git commit -m "short descriptive message about changes made here": associates a message with your commit
  • git push origin (place name of branch here without parentheses): sends the changes to github
